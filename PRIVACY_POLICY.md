## 1. Information We Collect

### 1.1 User Queries
- **What:** The text of your first-aid questions and queries
- **Why:** To process your request and provide first-aid guidance
- **How Long:** Queries are processed in real-time and not stored permanently

### 1.2 API Usage Data
- **What:** API calls made to Anthropic's Claude service
- **Why:** To generate responses to your queries
- **Third Party:** Your queries are sent to Anthropic (see Section 4)

### 1.3 Technical Information
- **What:** Browser type, IP address, session data, usage statistics
- **Why:** To operate and improve the Service, detect abuse
- **How Long:** Session duration or as required by law

### 1.4 API Keys
- **What:** Your Anthropic API key (if you provide one)
- **Why:** To authenticate API requests
- **Storage:** Stored in your browser session only (not on our servers)
- **Your Responsibility:** Keep your API key secure
